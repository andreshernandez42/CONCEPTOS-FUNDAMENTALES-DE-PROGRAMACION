# CONCEPTOS-FUNDAMENTALES-DE-PROGRAMACI-N
Este repositorio contiene el desarrollo de un sistema en Java orientado a la generación, procesamiento y análisis de información de ventas a partir de archivos en formato CSV. El proyecto incluye tanto la lógica de negocio implementada en Java como una propuesta de evolución hacia una interfaz web representada mediante un mockup en HTML.

La estructura del repositorio está organizada de la siguiente manera:

Clases Java:
Contienen la lógica principal del sistema, incluyendo la generación de datos (GenerateInfoFiles.java), el procesamiento de la información y la generación de reportes (Main.java).
Archivos CSV:
Archivos generados automáticamente que representan productos, vendedores y ventas por vendedor, utilizados como entrada para el sistema.
Reportes generados:
Archivos de salida (reporte_vendedores.csv y reporte_productos.csv) que consolidan la información procesada.
Mockup Web (HTML):
Representa una propuesta de interfaz gráfica para la evolución del sistema hacia una solución más interactiva y orientada al usuario.

El flujo del sistema inicia con la generación de datos de prueba, seguido por la lectura y procesamiento de la información, y finaliza con la creación de reportes organizados.

Este proyecto se encuentra en una fase preliminar, por lo cual su evolución contempla la integración con una interfaz gráfica funcional, el uso de bases de datos para persistencia de información y la implementación de mejoras en la arquitectura del software.
